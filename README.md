## 📌 Descripción del Proyecto

Aplicación de consola desarrollada en C# que simula el sistema de gestión de inventario de una tienda de tecnología.

El proyecto implementa los cuatro pilares de la Programación Orientada a Objetos (POO):

- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción
- Uso de Interfaces

---

## ⚙️ Funcionalidades

- Registrar Laptops
- Registrar Teléfonos
- Registrar Accesorios
- Aplicar descuentos automáticos según el tipo de producto
- Listar todos los productos
- Buscar producto por ID
- Calcular el total del inventario
- Generar reporte completo de productos

---

## 🧠 Estructura del Proyecto

- Producto (Clase abstracta base)
- Laptop, Telefono, Accesorio (Clases derivadas)
- IReportable (Interfaz)
- TiendaTech (Gestión de productos)
- Program (Menú interactivo)

---

## 🚀 Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `.slnx` o `.sln` en Visual Studio.
3. Compilar la solución.
4. Ejecutar con `Ctrl + F5`.
